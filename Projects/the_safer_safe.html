The Safer Safe

A secure safe with two-factor authentication by integrating fingerprint recognition and device-based unlocking

Overview

A homemade safe which requires two-factor authentication to be opened and accessed. This project was created for my grade 12 computer engineering final and it took many hours to complete. To open the safe it requires the registered fingerprint alongside an unlocking command from the users phone. There is a time limit to complete both checks and if only one check is completed, the safe will not open and it will reset. It was creating using an Arduino board alongside an ESP 8266 micro controller allowing communication via Wi-Fi.

Tools & Technologies

  • Arudinio
  • ESP8266 Microcontroller
  • C & C++
  • Circuit Design
  • Soldering
  • Manufacturing

Challenges

This was my first big project so everything was challenging as it was my first time really using all of the equipment to create something real and functional. During the development of this project, I was just learning how to use an Arduino board alongside an ESP8266 microcontroller so understanding how each thing worked and the nuances behind them was a big learning curve. Moreover designing and creating the circuit was also something that took me multiple attempts to get right as I was not familiar with these concepts before, especially soldering.
