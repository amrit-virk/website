MOSFET Based XOR Gate

The XOR logic gate created using MOSFETS

Overview

Similar to the power amplifier, this MOSFET-based XOR gate was also created entirely at home for the same class project. This design went a step further by requiring the logic gate to be constructed using MOSFET IC chips provided in our lab kits, rather than pre-built logic ICs. The objective was to produce a circuit that functioned identically to a standard XOR gate—outputting a HIGH voltage only when the two inputs differed. Once assembled, the gate’s behavior was tested using an AD3 system to ensure that the measured outputs matched the theoretical truth table for all input combinations.

Tools & Technologies

  • AD3
  • Wavelenghts
  • Circuit Design
  • Circuit Simulation

Challenges

This project required a strong understanding of how MOSFET’s behave and how they can manipulated to create an output that we desire, which in this case is that of an XOR gate. Like this power amplifier, this circuit also took a lot of trial and error in order to finally find a working combination. The main challenge was ensuring that the MOSFETs were correctly configured to achieve the desired logic gate behavior, as well as verifying that the output matched the expected truth table.
